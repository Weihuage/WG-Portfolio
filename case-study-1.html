Amazon Leo Console

UX Design · 2025–2026 · Amazon Kuiper

Overview

The Amazon Leo Console is the primary interface for maritime and mobility customers to monitor and manage their satellite connectivity. I led the UX design for the dashboard, service contracts, fleet management, and antenna monitoring experiences.

Challenge

Customers needed to manage hundreds of active connections across vessels and vehicles in real-time. The existing workflow required navigating between multiple disconnected views, leading to slow troubleshooting and missed outages.

Approach

  • Conducted stakeholder interviews and mapped existing workflows
  • Created information architecture for a unified dashboard
  • Designed a map-based status view with color-coded markers
  • Iterated through 3 rounds of prototype testing with internal users
  • Built interactive prototypes in React for design reviews

Outcome

The redesigned console reduced average troubleshooting time from 12 minutes to under 3 minutes. The unified dashboard became the default landing page for all Leo customers.

Key Decisions

  • Map-first design: geographic context is critical for maritime/mobility customers
  • Status color system: green (online), red (offline), grey (paused/cancelled) — tested for color-blind accessibility
  • Progressive disclosure: summary cards at top, detailed table below, drill-down on click
